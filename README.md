# emoji merge
这是一个小程序，用来对两个图像进行png融合，生成一幅图像。生成表情特点为：在白色背景下显示“表面图”，在黑色背景显示“隐藏图”。故而在微信，QQ等社交软件时，缩略图显示表面图，而点开大图后显示隐藏图。

### 注意：运行前一定要安装Pillow库！
该程序的运行（图片生成）依赖于PIL库，运行需安装之。
